Kitpages Cms Bundle
===================

Introduction
--------------

This bundle is used to manage editorial content in a website.

There is several readers for the documentation

* The PHP developper who needs to include this CMS into it's website
* The sysadmin who needs to configure apache and linux in order to make it run
* The designer who needs to build templates
* The user who wants to create pages and change content of it's site
